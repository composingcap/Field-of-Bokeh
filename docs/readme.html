Sliding Focus

Preface and documentation

Requirements

  • Max 8 and the Sliding Focus Max Project or the Slider Focus App
  • A stereo audio interface
  • A microphone for audio input
  • Four crotales pitched at D7, E7, G#7, and A7

    • Running the piece

    You may run Sliding Focus either with an external score or with the intiegrated score within the max project. The integrated score will turn pages for you at each phase of the piece.

    This piece makes changes without the use of explicit triggers. Scenes progress through each phase making subtle adjustment to the electronics while phases are articulated with a low swell. If you are using an external score, listen for these swells to move to a new phase.

    Whenever there is a phase change, the max patch will toggle the "transition armed" button and the phase transition will occur on the next onset.

    How the piece is tracked

    The piece will run itself based on both live input and a timer. The metrics used in calculating a transition score are the number of onsets and a persieved density of material. Here is a short description of how these work and are implemented.

    Timer: Counts time from the start of a phase.
    Onset Tracker: Attempts to track each hit of the crotales.
    Density: Uses a filtered amplitude tracker to see when the music has been consistantly loud or not.

    Some notes

  • When performing the piece make sure to let the electronic sounds inform your performance. The electronics will not be exactly the same each time as they are based on the instrumental input sounds.
  • Allow the music to breath and do not feel like you need to play all of the material on each page. The score is open and I want you to use it and the electonics
